# lab-module1-project1

Описание:

Проект создан на версии java 25.
В главном методе вызываются методы TransactionService,
который работает в режиме MODE1.

Необходимо:

Изменить код таким образом, чтобы можно было вызывать методы в разных режимах.
Например, сделать вначале вызов в режиме MODE1, затем в режиме MODE2.
У режимов один метод для старта, но разные методы завершения.
